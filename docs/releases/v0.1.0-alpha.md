# SilkMC v0.1.0-alpha

SilkMC begins as a compatibility-focused fork of Folia's latest stable upstream with its own branding, documentation, CI, and release flow.

## Goals

- preserve regional multithreading
- improve plugin compatibility pragmatically
- keep vanilla gameplay stable

## Compatibility targets

- Paper plugins
- Bukkit plugins
- Spigot plugins

## Roadmap

- deeper scheduler compatibility shims
- stronger thread violation diagnostics
- expanded benchmark coverage
- broader plugin compatibility validation

## Current limitations

- not all single-threaded plugin assumptions can be emulated safely
- some cross-region behavior remains intentionally strict
- published benchmark numbers are still pending broader validation
