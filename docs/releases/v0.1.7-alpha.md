# SilkMC v0.1.7-alpha

SilkMC v0.1.7-alpha is a small maintenance release that corrects a plugin-compatibility logging defect and tidies up the release tooling.

## Highlights

- Fix an inconsistent dedup key in the plugin compatibility warning tracker. The metadata path recorded the raw plugin name in `WARNED_PLUGINS` while the bytecode-inspection path recorded the normalized (trimmed, lowercased) name, so the same plugin could emit duplicate "not `silk-supported`" warnings and case variations of a name were treated as distinct. Both paths now normalize consistently.

## Maintenance

- Point the release workflow at the matching release notes (`body_path`) instead of the stale `v0.1.5-alpha` file.

## Notes

- No plugin compatibility behavior changed beyond the warning dedup fix; classification, enforcement, and override handling are unchanged.
- Region-threading and locator-bar behavior from `v0.1.6-alpha` are unaffected.
