# SilkMC v0.1.6-alpha

SilkMC v0.1.6-alpha restores the locator bar on the region-threaded server base and folds in a small set of stability-focused fixes derived from Canvas-style region-threading work.

## Highlights

- Re-enable `/waypoint` command registration on SilkMC's Folia-derived command set.
- Replace the stubbed waypoint manager path with a region-aware manager so locator bar tracking works again.
- Guard waypoint chunk visibility checks to the owning player tick thread.
- Preserve waypoint tracking correctly across same-region teleports.
- Fix respawn-anchor charge consumption in SilkMC's async respawn flow.

## Notes

- These changes are focused on core server stability and locator-bar behavior.
- No plugin compatibility codepaths were changed as part of this release.
